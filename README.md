# Geo Run

Mobile game concept powered by real-world activity data.

## Current direction

We are exploring a frontier settlement builder where a player's daily movement
feeds the growth of an outpost on a new land. The core fantasy is steady,
low-pressure expansion instead of tactical combat or a pure walking map.
