# Geo Run

Geo Run is a mobile game concept where real-world activity powers the growth of
a frontier settlement.

Players connect their activity data, turn movement into in-game resources, and
use those resources to expand a small outpost into a living settlement. The
core fantasy is quiet, visible progress: even an ordinary day should move your
town forward.

## Project Status

Geo Run is currently in the concept and pre-production stage.

The product direction has been defined, the MVP loop is being shaped, and the
team is exploring the visual language, rendering approach, and mobile-first
experience for the first playable version.

Current focus:

- define a clear MVP settlement-building loop;
- lock in the art direction for a cozy top-down frontier scene;
- validate the home screen and settlement presentation on mobile;
- keep scope narrow enough for a believable first playable build.

## What Comes Next

The next planned steps are:

1. Finalize the MVP feature set and progression structure.
2. Choose and integrate the first asset pack for the settlement scene.
3. Build the first asset-backed home screen for the mobile prototype.
4. Refine the technical architecture for rendering, game state, and content
   expansion.

## Vision

Geo Run aims to feel more like a game than a fitness tracker, and more
comforting than a challenge app. The long-term goal is a low-pressure mobile
settlement builder that turns everyday movement into a place that visibly grows
over time.
